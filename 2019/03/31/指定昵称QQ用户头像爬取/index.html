QQ音乐用户头像爬取

2019-03-31 技术 Python, shell, 图像拼接, 爬虫 评论

前言

  这篇文章记录一次心血来潮的操作。事情的起因是我要在QQ音乐网页端根据用户名查找用户，结果QQ音乐要我下载客户端才给我显示完整的查询结果。于是我就随手按下了F12,找到了搜索的接口，达成了我的目的。我看到返回的结果中有挺多头像链接，就开始想要用这些头像拼一个图。就有了下面的操作。

  需要用到的工具有：

  • bash(shell环境)
  • wget(用来发起请求)
  • jq(用来解析数据)
  • sed或awk(用来处理文本)
  • python + pillow(用来处理图像)

  1. w 指定搜索的关键词
